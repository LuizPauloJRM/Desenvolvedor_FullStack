Aula 10- Imagens

Para inserir uma imagem, utilizamos a tag <img>. É importante lembrar que essa tag não possui uma tag de fechamento.

O atributo src é utilizado para especificar o caminho da imagem que queremos exibir. O atributo alt é usado para fornecer uma descrição alternativa da imagem, que é importante para acessibilidade e SEO.
