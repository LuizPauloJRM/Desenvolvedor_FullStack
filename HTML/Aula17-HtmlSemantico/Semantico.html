Aula 17 - HTML Semantico

Elementos Semanticos

Elementos semânticos são aqueles que têm significado claro e específico, tanto para os desenvolvedores quanto para os navegadores. Eles ajudam a estruturar o conteúdo de forma lógica e acessível.

Exemplos de elementos semânticos incluem:

  • <header> - Define o cabeçalho do documento ou de uma seção.
  • <nav> - Define uma seção de navegação.
  • <main> - Define o conteúdo principal do documento.
  • <article> - Define um conteúdo independente, como um artigo ou post.
  • <section> - Define uma seção genérica dentro de um documento.
  • <footer> - Define o rodapé do documento ou de uma seção.
  • <aside> - Define conteúdo relacionado, mas não essencial ao conteúdo principal.
  • <figure> - Define conteúdo ilustrativo, como imagens ou gráficos, com uma legenda opcional.
