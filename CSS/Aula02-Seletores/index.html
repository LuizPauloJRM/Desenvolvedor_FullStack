Seletores em CSS

Vamos aprender a selecionar elementos HTML para aplicar estilos com CSS.

Existem vários tipos de seletores, como:

  • Seletores de tipo: selecionam todos os elementos de um tipo específico, como p ou h1.
  • Seletores de classe: selecionam elementos com uma classe específica, como .minha-classe.
  • Seletores de ID: selecionam um elemento com um ID específico, como #meu-id.
  • Seletores de atributo: selecionam elementos com um atributo específico, como [type="text"].
  • Seletores de pseudo-classes: selecionam elementos com base em seu estado, como :hover ou :focus.
  • Seletores de pseudo-elementos: selecionam partes específicas de um elemento, como ::before ou ::after.
